PhotoSorter Python script 2.1.0 Released

Published Monday, August 28, 2017 Reading Time ~2 min Filed Under dropbox вЂў python вЂў releases

Photosorter is a little Python script to keep my photos from DropboxВ organized.

It watches a source directory for modifications and moves new image files to a target directory depending on when the photo was taken, using EXIF data and creation date as a fallback. There is also an option to move existing photos.

Directory and file names follow a simple naming convention (YYYY-MM/YYY_MM_DD/YYYY-MM-DD hh:mm:ss.ext) that keeps everything neatly organized. Duplicates are detected and ignored based on their SHA1 hash and folder path. Photos taken in the same instant get de-duplicated by adding a suffix (-1, -2, etc) to theirВ filenames.
